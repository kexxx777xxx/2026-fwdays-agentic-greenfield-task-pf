# Ласкаво просимо до askdocs

Це **корпус за замовчуванням**, що монтується в контейнер як `/corpus`. Поклади
сюди свої `.md` файли — вони автоматично індексуються, і ти зможеш ставити по них
питання через CLI.

## Як спитати

```bash
docker compose run --rm app python -m askdocs.ingest   # проіндексувати /corpus
docker compose run --rm app python -m askdocs.cli "твоє питання"
```

## Приклад (домен ГущоЛіт)

Проєкт ГущоЛіт будує літаючі авто, двигун яких працює на переброджній кавовій
гущі. Це синтетичний приклад: відповідь можна дати лише з цього корпусу, а не із
загальних знань моделі.
