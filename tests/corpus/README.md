# ГущоЛіт

ГущоЛіт — проєкт літаючих автомобілів, що працюють на переробленій кавовій
гущі. Підйомну силу створюють чотири ротори-турбінки, які живляться від
гущепаливних елементів ГПЕ-7.

## Загальні характеристики

Крейсерська висота польоту ГущоЛіт-1 — 350 метрів. Максимальна дальність на
одному баку гущі — 420 кілометрів. Екіпаж — один пілот, до трьох пасажирів.

## Команда

Головний конструктор проєкту — Орися Кавунець. Відділ паливних систем очолює
Тарас Жмих.
